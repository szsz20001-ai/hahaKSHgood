# 💻 윈도우 ➡️ 맥북(macOS) 환경 이식 및 세팅 가이드

본 문서는 강승훈 과장의 기존 윈도우 데스크톱 업무 환경(일정 관리, 위젯 배치, 생산성 도구)을 맥북(macOS)으로 전환할 때, 최대한 이질감 없이 유사한 동선으로 구축하기 위한 이식 및 설정 가이드라인입니다.

---

## 1. 📅 바탕화면 및 일정 관리 세팅

현재 윈도우 바탕화면에 연동되어 있는 **DesktopCal**과 일정 서버를 맥북으로 마이그레이션하는 방법입니다.

### A. 바탕화면 달력 (DesktopCal)
* **공식 앱 설치**: DesktopCal은 macOS 버전을 공식 지원합니다.
  * [DesktopCal 공식 다운로드 페이지](https://www.desktopcal.com/)에서 Mac 버전을 설치합니다.
  * 기존 윈도우에서 로그인하여 사용 중이던 계정으로 로그인하면 일정 데이터가 즉시 연동됩니다.
* **대안 (Blotter 앱 - 유료)**: 
  * 맥의 순정 감성을 극대화하고, iCloud 캘린더와 연동하여 바탕화면에 투명하게 일정을 뿌리고 싶다면 Mac App Store의 **Blotter** 앱을 추천합니다.

### B. 기획부 일정 로컬 서버 (`editor_server.py`) 이식
파이썬 기반으로 제작된 서버이므로 맥북에서도 완벽하게 구동 가능합니다.
1. **파이썬 및 uv 환경 설치**:
   맥북 터미널(Terminal)을 열고 아래 명령어로 `uv`를 설치합니다:
   ```bash
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```
2. **경로 수정**:
   `editor_server.py` 내부의 윈도우 드라이브 경로(`E:\Shkang_Obsidian\...`)를 맥북의 로컬 경로(예: `/Users/jh/Shkang_Obsidian/...`)로 변경합니다.
3. **백그라운드 실행**:
   터미널에서 아래 명령어로 서버를 구동하면 동일하게 `http://localhost:8000`으로 일정 관리 페이지 접속이 가능합니다:
   ```bash
   uv run python editor_server.py
   ```

## 2. 🎛️ 바탕화면 레이아웃 및 위젯 구성

macOS의 데스크톱 위젯 기능을 활용해 기존 윈도우 우측 레이아웃(시계, 날씨, 단축 아이콘)을 동일하게 구성합니다.

* **데스크톱 폴더/아이콘 배치**: 
  * 기존 `Ksh_Work` 등의 폴더를 바탕화면 우측에 자유롭게 드래그하여 배치합니다.
* **아날로그 시계 위젯**:
  * 데스크톱 빈 곳을 우클릭하고 **[위젯 편집]**을 선택합니다.
  * '시계' 카테고리에서 아날로그 스타일 시계를 우측 상단에 배치합니다.
* **날씨 위젯**:
  * '날씨' 카테고리에서 현재 위치(당하동) 기반의 사각형 위젯을 선택해 아날로그 시계 아래에 배치합니다.
* **디지털 시계**:
  * 큼직한 시계 텍스트를 띄우기 위해 App Store에서 **MD Clock**이나 **Flip Clock** 같은 무료 위젯 앱을 활용해 우측 하단에 대형 시계를 배치합니다.

## 3. 🛠️ 생산성 도구 이식 (PowerToys 대체)

윈도우에서 사용하던 PowerToys 기능들은 맥의 기본 기능과 훌륭한 무료 서드파티 앱들로 완벽하게 대응합니다.

| 윈도우 기능 (PowerToys) | macOS 매칭 기능 / 앱 | 설정 및 설명 |
| :--- | :--- | :--- |
| **Peek** (미리보기) | **Quick Look** (순정 기본 내장) | 파일을 선택하고 **`Space`** 키를 누르면 사진/영상/문서가 즉시 미리보기됩니다. |
| **PowerToys Run** (빠른 실행) | **Raycast** (강력 추천, 무료) | `Alt` + `Space` 대신 맥에서는 Raycast를 설치해 시스템 제어, 앱 실행, 계산기 등으로 활용합니다. |
| **FancyZones** (창 분할) | **Rectangle** (무료) 또는 Sequoia 순정 기능 | 최신 macOS Sequoia는 창 분할을 기본 지원하며, 더 세밀한 분할은 **Rectangle** 앱을 권장합니다. |
| **Awake** (화면 켜짐 유지) | **Amphetamine** (추천, 무료) | 맥 App Store에서 설치하며, 메뉴바에서 한 번의 클릭으로 화면 꺼짐을 제어합니다. |

## 🚀 맥북 구매 당일 세팅 추천 로드맵

1. 맥북 초기 설정 완료 후 **App Store** 접속
2. 필수 무료 생산성 앱 다운로드:
   * **Amphetamine** (화면 켜짐 유지)
   * **Rectangle** (창 분할 단축키 설정)
3. 브라우저에서 아래 사이트 방문 및 설치:
   * [Raycast](https://www.raycast.com/) (빠른 실행창)
   * [DesktopCal](https://www.desktopcal.com/) (바탕화면 달력)
4. 기존 Obsidian 보관소(`Shkang_Obsidian`) 폴더를 맥북의 `문서` 또는 `사용자` 폴더 밑으로 통째로 복사
5. 옵시디언 설치 후 복사한 보관소 폴더 열기 (기존 메모와 회의록, 데일리 노드가 100% 보존됩니다.)
6. 일정 서버 경로 조정 후 파이썬 백그라운드 구동 테스트
