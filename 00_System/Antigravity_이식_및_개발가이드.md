# 🚀 Antigravity 이식 및 AI 개발 협업 가이드

이 문서는 다른 컴퓨터에 현재 개발된 시스템(캘린더, 비디오 자동 분류 스크립트, 평면도 뷰어 등)을 이식하고, 향후 AI와 협업할 때 어떤 방식으로 프롬프트를 작성하고 문제를 해결했는지에 대한 기록입니다.

---

## 1. 📅 캘린더 & 스케줄러 시스템 이식 가이드

회의록과 일정을 로컬에서 연동하고 저장하기 위한 Flask 웹 서버 및 캘린더 프론트엔드 시스템입니다.

### 💻 다른 컴퓨터로 이식 방법
1. **Python 설치 및 가상환경(venv) 구축**:
   - Python 3.10 이상 버전을 설치합니다.
   - 프로젝트 폴더(예: `D:\Ksh_work\01.계획`)에서 터미널을 열고 가상환경을 생성합니다.
     ```bash
     python -m venv venv
     .\venv\Scripts\activate
     ```
2. **필수 라이브러리 설치**:
   - Flask 서버 구동을 위해 다음 패키지를 설치합니다.
     ```bash
     pip install Flask Flask-Cors
     ```
3. **데이터베이스 백업 및 이동**:
   - 기존 컴퓨터의 `D:\Ksh_work\01.계획\calendar.db` (SQLite 파일)을 복사하여 새 컴퓨터의 동일한 폴더에 넣습니다. 이 파일에 모든 캘린더 할 일, 체크박스 상태, 일정 데이터가 보관되어 있습니다.
4. **로컬 서버 실행**:
   - 백그라운드 서버인 `editor_server.py`를 실행합니다.
     ```bash
     python editor_server.py
     ```
   - 서버는 기본적으로 `http://localhost:5000`에서 대기하며, `캘린더_인쇄용.html`이 실행될 때 API를 통해 이벤트를 읽고 씁니다.

### ⚠️ 주요 시행착오와 해결 방법 (Troubleshooting)
- **정적 파일 캐싱 문제**:
  - **문제**: 기존에는 브라우저가 `events.js`를 캐싱하여, 로컬 스토리지나 파일이 수정되어도 페이지를 새로고침하면 이전 상태로 되돌아가는 현상이 발생했습니다.
  - **해결**: `editor_server.py`에 `/api/load_events` GET 라우트를 추가하고 JSON 데이터로 동적 로드하도록 고쳤습니다. `캘린더_인쇄용.html`에서 호출 시 타임스탬프(`?t=Date.now()`)를 붙여 캐시를 원천 차단했습니다.
- **업무 자동 이월(Carry-over) 및 렌더링 최적화**:
  - 미완료된 할 일은 다음 날로 자동 이월하되, 완료(`[x]`)된 항목은 취소선과 함께 진행률 표시줄(Progress bar)이 연동되도록 DOM 렌더링 로직을 수정했습니다.

## 2. 🎬 비디오 자동 분류 및 정렬 시스템 (`organize_video_files.py`)

189개의 대용량 원본 비디오 파일(A7C, 드론, A7M4 등)을 촬영 동선 및 편집 순서 시퀀스(Seq 01~26)에 맞춰 즉각 정리하는 스크립트입니다.

### 💻 다른 컴퓨터로 이식 방법
1. **필수 패키지 설치**:
   - 영상 메타데이터 분석(촬영 시간 등)을 위해 OpenCV와 NumPy가 필요할 수 있습니다.
     ```bash
     pip install opencv-python numpy
     ```
2. **스크립트 실행**:
   - 스크립트(`organize_video_files.py`) 내의 원본 경로(`03.원본`)와 대상 경로(`편집순서_정리`)를 새 컴퓨터의 드라이브 경로에 맞게 수정한 후 실행합니다.
     ```bash
     python organize_video_files.py
     ```

### ⚠️ 주요 시행착오와 해결 방법 (Troubleshooting)
- **복사(Copy)와 이동(Move)의 성능 차이**:
  - **문제**: 기가바이트(GB) 단위의 영상들을 복사하면 디스크 공간이 부족해지고 시간이 수십 분 이상 걸리는 치명적인 문제가 있었습니다.
  - **해결**: 동일한 드라이브 내에서는 파일 메타데이터 레코드만 변경되므로 복사가 아닌 `shutil.move`를 사용하여 **1초 만에 즉시 정리**가 완료되도록 최적화했습니다.

## 3. 🗺️ 평면도 및 미니맵 SVG 렌더링

`촬영기획서_큐시트.html` 내의 미니맵 및 카메라 짐벌 이동 동선 시퀀스 시각화 시스템입니다.

### ⚠️ 주요 시행착오와 해결 방법 (Troubleshooting)
- **LaTeX Math 파서와 코드 충돌**:
  - **문제**: HTML 및 Javascript 코드 중 마크다운 그리드나 강조 문자를 처리하기 위해 정규식 패턴 `/\\*\\*(.*?)\\*\\*/g` 혹은 `**`를 그대로 작성했을 때, Antigravity AI 시스템의 프론트엔드 파서가 이를 LaTeX 수식의 분수(`\frac{`) 기호로 오인하여 웹페이지 내부의 SVG 그리드 렌더링 스크립트 자체가 깨져버리는 특이한 버그가 발생했습니다.
  - **해결**: 스크립트 내에서 리터럴 정규식 슬래시(`/`) 대신 문자열 생성자인 `new RegExp('\\\\*\\\\*(.*?)\\\\*\\\\*', 'g')`를 사용하여 시스템 파서의 필터링을 우회하여 정상 작동하도록 해결했습니다.

## 4. ✍️ AI 프롬프트 작성 템플릿 및 협업 요령

향후 다른 컴퓨터나 새로운 대화 세션에서 AI(제미나이, 클로드 등)에게 업무를 시킬 때 실패 확률을 줄이는 실전 프롬프트 예시입니다.

### 📌 A. 도면 및 평면도 수정 요청 시
> **프롬프트 템플릿:**
> ```text
> [구조 지정]: 이번에 작업할 건물은 [ㅁ자 / ㄷ자 / ㄴ자] 닫힌/열린 구조이다.
> [공간 배치 상세]: 현관을 기준으로 왼쪽에는 [거실/중정], 오른쪽 복도를 따라 [침실1 -> 욕실 -> 침실2] 순으로 배치되어 있다.
> [세부 예외사항]: 기존 도면과 달리 보일러실은 없으며, 대신 세탁실이 크게 위치한다. 침실1과 욕실은 분리되어 있고 그 사이를 복도가 연결한다.
> [창문 및 개구부]: 모든 방 and 복도의 외벽 부분에는 창문(Window) 표시를 SVG 상에 명확히 구분할 수 있도록 얇은 이중선이나 하늘색 사각형으로 렌더링해 줘.
> ```

### 📌 B. 비디오 분류 및 분석 요청 시
> **프롬프트 템플릿:**
> ```text
> 1. 원본 폴더 내 파일명 패턴(예: DSC001.mp4, C0001.MP4 등)을 읽고 촬영 시간순으로 정렬해 줘.
> 2. 촬영 대상 동선인 [외관 -> 현관 -> 중문 -> 복도 -> 침실2 -> 욕실 -> 침실3 -> 거실 -> 중정 -> 주방 -> 세탁실 -> 드레스룸 -> 욕실1 -> 침실1 -> 중정 -> 현관] 순서에 따라 폴더를 순서대로 넘버링(Seq 01 ~ Seq 26)하여 생성해 줘.
> 3. 파일 복사는 용량을 많이 먹고 느리니, shutil.move를 이용해 즉시 이동하는 파이썬 코드를 작성하고 실행해 줘. 빈 폴더가 생기는 경우 플레이스홀더 텍스트 파일을 넣어 구조를 유지해 줘.
> ```

### 📌 C. 캘린더/서버 기능 개선 요청 시
> **프롬프트 템플릿:**
> ```text
> 1. Flask 서버(`editor_server.py`)의 포트는 5000번을 유지하고, 클라이언트(`캘린더_인쇄용.html`)에서 일정을 완료처리하면 즉시 SQLite `calendar.db`에 동기화되게 해 줘.
> 2. 브라우저 캐싱으로 인해 새로고침 시 데이터가 옛날 정보로 롤백되는 현상을 막기 위해, 로컬 스토리지 대신 API 통신 시 캐시 버스팅 파라미터(?t=timestamp)를 사용하도록 JS 코드를 수정해 줘.
> 3. 업무 완료 상태(`[x]`)는 텍스트에 취소선을 긋고 배경색을 흐리게 처리하며, 미완료 상태(`[ ]`)는 오늘 날짜가 지나면 자동으로 다음 날로 복사(이월)되도록 백엔드 DB 저장 시 날짜값을 처리해 줘.
> ```
