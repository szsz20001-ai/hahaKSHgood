# 🗺️ 건축 도면 보는 법 — 기초 가이드

> **작성일:** 2026-06-12  
> **분류:** 촬영 기획 기초 지식  
> **목적:** 주택/인테리어 촬영 기획 시 도면을 정확히 읽고 동선·마감재·공간 구조를 파악하기 위한 참고서

---

## 🔑 핵심 한 줄 요약

> **"벽은 굵게, 문은 반원, 치수는 mm"** — 이 3가지만 기억해도 도면의 60%는 읽힌다.

---

## 1️⃣ 평면도란 무엇인가?

평면도(Floor Plan)는 건물의 바닥에서부터 **약 1.2~1.5m 높이를 수평으로 잘라 위에서 아래로 내려다본 그림**이다.  
방의 배치, 벽체, 문, 창문, 가구의 위치 등을 한눈에 파악할 수 있는 **'공간 설명서'** 역할을 한다.

### 도면의 종류

| 도면 종류 | 설명 | 촬영 기획 활용도 |
|----------|------|----------------|
| **평면도** | 위에서 수평으로 자른 단면 | ⭐⭐⭐ 동선 파악, 공간 구성 |
| **입면도** | 건물을 옆에서 본 모습 | ⭐⭐ 외관, 창문 높이 파악 |
| **단면도** | 건물을 수직으로 잘라 본 모습 | ⭐⭐ 층고, 천장 높이 파악 |
| **상세도** | 특정 부위를 확대한 도면 | ⭐ 마감재 디테일 확인 |

---

## 2️⃣ 도면 읽기 4단계 (실전 순서)

```
1. 방위(N 화살표) 확인
   → 북쪽이 어디인지 파악 = 채광, 햇빛 방향 추정

2. 전체 벽체 구조 파악
   → 굵은 선(내력벽) vs 얇은 선(비내력벽) 구분
   → 공간의 큰 틀 이해

3. 범례(Legend) 확인
   → 도면 하단/측면의 기호 설명표 먼저 읽기
   → 도면마다 기호가 조금씩 다름

4. 치수 & 마감재 표기 확인
   → 숫자는 mm 단위
   → 마감재 약어(TL, WD 등) 파악
```

---

## 3️⃣ 선(Line) 종류와 의미

| 선 종류 | 의미 | 실무 포인트 |
|---------|------|------------|
| **굵은 실선** | 내력벽 (철거 불가) | 공간 구조의 뼈대, 절대 변경 불가 |
| **얇은 실선** | 비내력벽, 치수선, 마감선 | 공간 분할용, 구조 변경 가능 |
| **점선 (- - -)** | 숨은 구조물, 붙박이가구, 천장 구조 | 눈에 보이지 않는 요소 |
| **일점 쇄선** | 중심선, 기준선 | 공간의 기준점 표시 |
| **이점 쇄선** | 경계선, 가상선 | 영역 구분 |

---

## 4️⃣ 문(Door) & 창문(Window) 기호

### 문 기호 종류

```
여닫이문 (가장 흔함)
┌────────
│       ╲ ← 반원 모양 = 문이 열리는 궤적
│        ╲

슬라이딩문 (미닫이)
│══════════│ ← 문짝이 옆으로 밀리는 방향 표시
←──────────

접이식 도어 (폴딩)
│╱╲╱╲╱╲│ ← W 모양
```

### 창문 기호

```
일반 창문 (외벽 위치)
──┬──
  │  ← 두 줄 = 이중창
──┴──

고정창 (FIX)
──═══── ← 단순 실선 (열리지 않음)
```

### ⚠️ 촬영 기획 시 문 위치 확인 포인트
- **여닫이문:** 어느 방향으로 열리는지 반원으로 확인 → 짐벌 진입 경로 계획
- **슬라이딩 중문:** 열리는 방향 확인 → REVEAL Shot 방향 결정
- **문 위치가 공간 첫 인상을 결정한다** — 카메라가 문을 통과할 때의 구도가 핵심

## 5️⃣ 치수(Dimension) 읽기

### 기본 단위
> 건축 도면의 **모든 수치는 mm(밀리미터)** 가 기본

```
도면 표기 → 실제 크기
  2000   →   2m (2,000mm)
  1200   →   1.2m
   900   →   0.9m (일반 문폭)
  2400   →   2.4m (표준 천장고)
```

### 자주 등장하는 치수

| 치수 | 의미 | 촬영 관련 |
|------|------|----------|
| **2,400~2,800** | 일반 실내 천장고 | 14mm 광각으로 충분 |
| **1,200~1,500** | 오픈 천장/거실 높은 천장고 | Tilt-Up 샷 필요 |
| **900** | 표준 문 너비 | 짐벌 통과 가능 |
| **700~800** | 좁은 복도/욕실 문 | 짐벌 세로 방향 필수 |
| **1,100** | 창문 설치 높이 (이번 현장) | 창문 통해 외부 촬영 시 창틀 노출 주의 |

### 치수 확인 원칙
- ✅ **도면 수치 우선** — 자로 재는 것은 오차의 원인
- ✅ **치수선(←→)** 사이의 숫자가 실제 길이
- ✅ 외벽 치수는 벽 중심선 기준, 내벽은 마감면 기준인 경우 多

## 6️⃣ 마감재 표기 방법

### 주요 약어 코드

| 약어 | 영문 | 의미 |
|------|------|------|
| **TL** | Tile | 타일 |
| **ST** | Stone | 석재 |
| **WD** | Wood | 목재/마루 |
| **WP** | Wall Paper | 벽지 |
| **PT** | Paint | 도장/페인트 |
| **GL** | Glass | 유리 |
| **MT** | Metal | 금속 |
| **VT** | Vinyl Tile | 데코타일 |
| **SH** | Sheet | 시트지/필름 |

### '지정 마감' 표기 해석

```
도면 표기: "지정 강마루 마감"
→ 별도 실내재료마감표에서 구체적 제품명/색상 확인 필요
→ 예: 구정마루 마뷸러스 젠 모로칸크림

도면 표기: "지정 타일 마감"
→ 타일 규격(600각, 1200각 등) + 색상 코드 확인
→ 예: 라보나 1200 + 디딤블 패턴
```

---

## 7️⃣ 도면에서 자주 쓰는 기타 기호/약어

| 기호/약어 | 의미 |
|---------|------|
| **N (↑)** | 북쪽 (방위) |
| **W + 숫자** | 폭(Width) 표기 (예: W900 = 폭 900mm) |
| **H + 숫자** | 높이(Height) 표기 (예: H:1,100 = 높이 1,100mm) |
| **@ + 숫자** | 간격(Spacing) (예: @400 = 400mm 간격) |
| **FL** | Floor Level — 마감 바닥 높이 |
| **UP / DN** | 계단 올라감 / 내려감 방향 |
| **AD / PD** | 공기덕트 / 배관덕트 |
| **FIX** | 고정 (열리지 않는 창/문) |
| **오픈 (OPEN)** | 벽 없이 열려 있는 개구부 |
| **APP.** | 지정(Appointment) — 별도 마감표 참조 |

---

## 8️⃣ 도면 보면서 실수하는 것들 (주의사항)

### ❌ 흔한 실수들

1. **문 열리는 방향을 반대로 이해**
   → 반원의 호가 시작되는 쪽이 문 고정점(힌지), 끝나는 쪽이 손잡이

2. **치수를 자로 재서 환산**
   → 도면은 인쇄 크기와 실제 크기가 다름. 반드시 표기된 숫자 사용

3. **점선을 무시**
   → 점선은 숨어있는 중요 구조물 (천장 구조, 붙박이장 위치 등)

4. **범례를 안 봄**
   → 도면마다 기호가 다를 수 있음. 범례 없이 가정하면 오독 가능

5. **평면도만 봄**
   → 입면도·단면도를 함께 보면 천장 높이, 창문 높이 등 입체 정보 파악 가능

## 9️⃣ 영상 촬영 기획에 도면 활용하는 법

### 도면 = 촬영의 내비게이션

```
촬영 전 도면 체크 순서:
1. 공간 동선 파악 → 짐벌 워킹 경로 계획
2. 문 위치 확인 → REVEAL Shot 포인트 설정
3. 창문 위치/높이 → 자연광 방향 예측
4. 마감재 위치 → 포커스 촬영 포인트 선정
5. 좁은 공간(욕실/복도) → 렌즈 선택 & 짐벌 그립 방식 결정
```

### 도면을 영상에 활용하는 방법

| 활용 방법 | 효과 |
|----------|------|
| **도면 오버레이 (우측 하단)** | 시청자가 현재 위치 파악 가능 |
| **도면 → 실제 공간 컷 연결** | 설계 의도의 구현 증명 |
| **도면 위 동선 애니메이션** | 인트로에서 전체 구조 소개 |
| **마감재 도면 확대 → 클로즈업** | 디테일 전문성 강조 |

### 이번 현장(부여 오수리)에서 틀렸던 부분 ⚠️

> 처음 작성한 미니맵 동선이 실제 도면과 달랐던 이유:

- **복도 폭을 무시** → 짐벌이 통과할 수 없는 경로 계획
- **문 위치를 대략적으로만 표시** → 여닫이 방향 오류로 동선 불일치
- **중정 진입 개구부 위치 오독** → 복도1에서 중정으로 가는 실제 입구와 다름
- **세탁실·보일러실 위치** → 도면 좌상단에 있는데 주방 다음 이어지는 구조임을 늦게 파악

### ✅ 다음 촬영 기획 때 개선할 점

- [ ] 현장 방문 전 도면 출력해서 문 위치에 ○ 표시
- [ ] 좁은 공간 치수를 mm로 계산해 짐벌 통과 가능 여부 사전 확인
- [ ] 도면에 촬영 동선을 직접 손으로 그려보기 (빨강 펜으로 번호 표시)
- [ ] 마감재 약어 목록을 미리 별도로 정리해서 가지고 가기
